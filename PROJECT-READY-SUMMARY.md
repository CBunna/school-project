## 🔧 Troubleshooting Quick Reference

### Backend won't start
```bash
docker ps                    # Check if PostgreSQL is running
docker start beskydy-postgres
cd backend && npm install    # Reinstall dependencies
```

### Database errors
```bash
cd backend
npm run db:setup  # Reset database with fresh data
```

### Frontend issues
```
- Clear browser cache
- Clear localStorage (DevTools → Application → Storage)
- Check browser console for errors
```
